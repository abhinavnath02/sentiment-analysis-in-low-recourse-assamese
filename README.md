# Sentiment Analysis in Low-Resource Assamese using IndicBERT

## Project Overview
This project aims to build a high-quality, ethically collected Assamese sentiment dataset from public online sources to fine-tune IndicBERT. This is a research pipeline focused on reproducibility, ethics, and transparency.

## Phases
1. **Public Data Collection**
2. **Text Filtering, Cleaning, and Deduplication**
3. Manual Annotation
4. Model Training and Evaluation

## Reproducibility
- No Selenium/Puppeteer (requests-based only)
- No API keys required
- Anonymization by design
